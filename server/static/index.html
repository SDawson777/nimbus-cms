Previewing & Studio

To preview drafts, add ?preview=true&secret=YOUR_PREVIEW_SECRET or set the header X-Preview: true and X-Preview-Secret.

To open the Sanity Studio (content editing), run the Studio locally with npm run dev:studio or visit your deployed Studio. The API and Studio share the same Sanity project/dataset configuration.

Notes for integrators

  • Mobile base URL should point to this server; mobile expects `/content/*` endpoints.
  • Legacy endpoints are available under /api/v1/content/* for backwards compatibility.
